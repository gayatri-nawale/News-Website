# News App

A simple React + Vite application to display top news using the NewsAPI.

## 🚀 Getting Started

### 1️⃣ Clone the repository
```bash
git clone <repo-url>
cd News

```

#install dependencies
```bash
npm install
```
## Create .env file 
```bash
VITE_NEWS_API=your_api_key_here
```
## run 
```bash
npm dev run 
```
